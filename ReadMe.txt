програма:
1. выбрать файл CSV по кнопке 
	с учетом последнего файла (взять из settings.json)
	сщхранить выбраны путь в settings.json
	в меню должен быть фильтр по типу файла ("csv")

2. открыть файл csv как текст
		убрать шапку (пока отсчитав нужное количество строк до заголовка)
		сохранить как текст

3. открыть как CSV
		получить из заголовка номера столбцов для -> (взять из настроек - "columns")
	
4. создать footprint.json
	получить из settings.json columns ("Footprint","Rotation","FeedT")
	для каждой строки без повторений:
	{
	      "0402С": {
	         "Rotation": 0.0,
	         "FeedT": "CL8-2",
	         },
	      "0402R": {
	         "Rotation": 90.0,
	         "FeedT": "CL8-2",
	         },

	}
